# Nuxt train app

## Covered topics:

 - create own layout
 - use middleware
 - routing in nuxt
 - server side rendering
 - nuxt validation   
 - vuex with nuxt 

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
